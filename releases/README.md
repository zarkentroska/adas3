# Releases

Esta carpeta contiene los ejecutables compilados del sistema ADAS3.

## Estructura

- `server/` - Ejecutables del servidor ADAS3
- `client/` - Ejecutables del cliente ADAS3

Cada subcarpeta contiene ejecutables para:
- `windows/` - Archivos .exe
- `linux/` - Paquetes .deb y otros formatos
- `android/` - Archivos .apk
