# ADAS3 - Releases

Repositorio de ejecutables y releases del sistema ADAS3.

**Versión: v0.5 Alpha**

## Descripción

Este repositorio contiene los ejecutables compilados del sistema ADAS3 para diferentes plataformas:

- **Windows**: Archivos `.exe`
- **Linux**: Paquetes `.deb` y otros formatos
- **Android**: Archivos `.apk`

## Estructura

```
releases/
├── server/
│   ├── windows/    # Ejecutables .exe del servidor
│   ├── linux/      # Paquetes .deb y otros del servidor
│   └── android/    # Archivos .apk del servidor
└── client/
    ├── windows/    # Ejecutables .exe del cliente
    ├── linux/      # Paquetes .deb y otros del cliente
    └── android/    # Archivos .apk del cliente
```

## Repositorios de Código Fuente

Para ver o contribuir al código fuente, visita:

- **[ADAS3 Server](https://github.com/zarkentroska/ADAS3-Server)** - Código fuente del servidor
- **[ADAS3 Client](https://github.com/zarkentroska/ADAS3-Client)** - Código fuente del cliente

## Descargas

Los ejecutables estarán disponibles en las carpetas correspondientes según la plataforma y versión.

## Notas

- Los ejecutables se actualizan con cada nueva versión
- Consulta los repositorios de código fuente para reportar bugs o solicitar características
- Las versiones se etiquetan siguiendo el formato de versionado semántico
